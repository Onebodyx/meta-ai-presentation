# Chess Coach (MVP) — локальный каркас

Учебный помощник: игра против движка и пояснения от облачного LLM по данным **Stockfish**. На этом этапе — минимальная страница **Streamlit**, чтение настроек из `.env` и **узкий слой Cursor Cloud** (проверка ключа без сценария объяснений ходов).

## Быстрый старт (Windows, PowerShell)

Из корня репозитория, с уже созданным виртуальным окружением `.venv`:

```powershell
Copy-Item .env.example .env
# Отредактируйте .env: путь к Stockfish, ключ и при необходимости параметры Cursor Cloud
.\.venv\Scripts\Activate.ps1
streamlit run app.py
```

Одна явная команда запуска после активации venv: `**streamlit run app.py**`.

## Cursor Cloud (Cloud Agents API)

Интеграция зафиксирована в модуле `[cursor_cloud.py](cursor_cloud.py)`: HTTP **Basic** (ключ как имя пользователя, пароль пустой), см. [документацию Cursor](https://cursor.com/docs/cloud-agent/api/endpoints).


| Переменная                    | Назначение                                                                                                                                                                   |
| ----------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `CURSOR_CLOUD_AGENTS_API_KEY` | API key из Cursor Dashboard → **Cloud Agents** → **My Settings** → API keys.                                                                                                 |
| `CURSOR_CLOUD_BASE_URL`       | Базовый URL без завершающего слэша. Если пусто, используется `https://api.cursor.com`.                                                                                       |
| `CURSOR_CLOUD_MODEL`          | Необязательно. Пусто, `default` или не задано — поле `model` в будущих JSON-запросах **не отправляется** (модель выбирает Cursor Cloud). Иначе — явный идентификатор модели. |
| `CURSOR_CLOUD_TIMEOUT_SEC`    | Таймаут HTTP в секундах; если пусто или некорректное значение — **30**.                                                                                                      |


### Проверка готовности подключения (без Streamlit и без объяснений ходов)

После активации venv и заполнения `.env`:

```powershell
python cursor_cloud.py
```

При успехе выводится краткая информация из `GET /v0/me`; при ошибке — диагностическое сообщение (ключ, сеть, таймаут, HTTP-код).

В интерфейсе Streamlit можно нажать **«Проверить подключение к Cursor Cloud»** на той же странице, где отображается статус переменных.

## Переменные `.env` (остальное)


| Переменная       | Назначение                                                      |
| ---------------- | --------------------------------------------------------------- |
| `STOCKFISH_PATH` | Путь к исполняемому файлу Stockfish (например `stockfish.exe`). |


Секреты не хранятся в коде; `python-dotenv` подхватывает `.env` при импорте `[settings.py](settings.py)`.

## Зависимости

Runtime-зависимости — в `[requirements.txt](requirements.txt)`. Для HTTP к Cursor Cloud используется **httpx**.

## Вручную до следующего этапа

Установите или соберите **бинарник Stockfish**, убедитесь, что путь в `STOCKFISH_PATH` верный. Для облачной части подготовьте **Cursor Cloud** по таблице выше и проверьте подключение командой `python cursor_cloud.py`.
